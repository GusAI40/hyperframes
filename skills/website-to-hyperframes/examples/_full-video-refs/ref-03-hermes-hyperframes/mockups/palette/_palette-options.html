HERMES × HYPERFRAMES — PALETTE OPTIONS

Four palette directions grounded in 90s analog hacker / mystical color theory. All use #000 as ground, all are desaturated (~65–80%), all avoid pure RGB primaries that read "2016 web." Each card shows the same hero beat (HYPERFRAMES wordmark) in that palette so you can compare at a glance.

A · SODIUM TERMINAL

Theory: monochromatic warm — single hue family at varying values. Like a CRT showing nothing but amber, sodium-vapor street lamp at 3am.
Reference: Alien Nostromo terminals · early Apple ][ amber phosphor · 1985 IBM monochrome.
Cluster differentiation: via density / ASCII texture / motion type — never via hue.

WHAT TO REACT TO

  • A · Sodium terminal — most disciplined. Cluster variety has to come 100% from animation/density. The whole video reads as ONE artifact viewed under one light source.
  • B · Phosphor archive — like A but cool. Same constraint. The "decayed" green is the key — it's not the cyberpunk-revival neon, it's the actual oxidized phosphor of an old CRT.
  • C · Blade Runner (split complement) — most cinematic. Two hues sit in tension across the wheel. Clusters split warm vs cool. Highest mood ceiling.
  • D · Cyberpunk archive (triadic) — most variety while still controlled. Three muted anchor hues for the three "feature tour" cluster groups. Lottie cluster gets its triptych identity back without the V1 polychrome cheap.

NOTES ON COLOR THEORY APPLIED

  • All four palettes use OKLCH-equivalent ~70% saturation max — never fully saturated RGB primaries. That's the "muted 90s analog" thing.
  • All four use a slightly tinted highlight color (warm bone or cool bone) instead of pure #FFFFFF — which reads "digital" / 2016.
  • Structural color sits between bg and mid — gives terminal panels and code blocks weight without becoming visible noise.
  • Chromatic aberration on hero text uses real R/B values (e.g. rgba(220,40,80,0.45)) but at 1–3px offset — subtle, period-correct.

WHICH ONE

  • If you want most cinematic, pick C.
  • If you want most disciplined / one mood, pick A or B.
  • If you want the cluster-color-identity from V1 done right, pick D.
